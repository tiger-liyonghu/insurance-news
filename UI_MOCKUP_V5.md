## 设计细节

### 1. Header (顶部栏)

**布局**：
- 左侧：标题 + 副标题
- 右侧：系统状态指示灯

**状态指示灯**：
- 🟢 Agent Live - 正常运行（绿色脉冲动画）
- 🟡 Agent Pending - 等待中（黄色）
- 🔴 Agent Offline - 离线（红色）

**样式**：
- 深色渐变背景（#0052CC → #1A1C1E）
- Sticky 定位（滚动时保持可见）
- 状态指示灯有脉冲动画效果

---

### 2. Sidebar (左侧筛选器)

**参考 Chainalysis 的多维过滤系统**：

#### 全球地区筛选
- 多选复选框（Checkbox）
- 支持全选/全不选
- 实时联动更新主内容区

**选项**：
- ☑ 美国 (US) 🇺🇸
- ☑ 英国 (GB) 🇬🇧
- ☐ 中国 (CN) 🇨🇳
- ☐ 泰国 (TH) 🇹🇭
- ☐ 新加坡 (SG) 🇸🇬
- ☐ 马来西亚 (MY) 🇲🇾
- ☐ 阿联酋 (AE) 🇦🇪
- ☐ 沙特 (SA) 🇸🇦

#### L&H 业务线筛选
- 多选复选框
- 实时联动

**选项**：
- ☑ 寿险
- ☑ 医疗
- ☐ 重疾

#### 统计信息
- 总案例数
- 筛选后案例数
- 实时更新

---

### 3. Main Content (主内容区)

#### Intelligence Dashboard (数据概览)
- 3 个指标卡片：
  - 监控国家数
  - 累计案例数
  - 最近更新时间

#### Case Intelligence (案例卡片流)

**卡片设计（参考 CAIF 简报风格）**：


**卡片样式**：
- 白色背景，12px 圆角
- 微阴影效果
- Hover 时轻微上浮
- 头部：深色背景，白色文字
- 中部：加粗的 MO 描述（红色强调）
- 底部：浅蓝色背景的专家启示容器

**数据流控（参考 Sentry）**：
- 首页只显示 Top 10 个案例
- 其余案例折叠在 Expander 中
- 筛选条件下依然保持 Top 10 原则

### 4. 响应式设计

- 移动端：侧边栏自动折叠
- 平板：侧边栏可切换显示/隐藏
- 桌面：侧边栏始终显示

## 颜色方案

- **主色调**：#0052CC (科技蓝)
- **深色**：#1A1C1E (深空灰)
- **浅色**：#F8F9FA (极简浅灰)
- **强调色**：#DC3545 (红色，用于 MO 标题)
- **成功色**：#28A745 (绿色，用于状态指示)
- **警告色**：#FFC107 (黄色，用于状态指示)

---

## 动画效果

1. **状态指示灯**：脉冲动画（2秒循环）
2. **卡片 Hover**：轻微上浮 + 阴影加深
3. **筛选联动**：平滑过渡（0.3s）
4. **数据加载**：Spinner 动画

---

## 交互逻辑

1. **实时筛选**：选择筛选条件后，主内容区立即更新
2. **Top 10 原则**：无论筛选条件如何，首页只显示前10个
3. **Expander 折叠**：第11个及以后的案例自动折叠
4. **状态更新**：每30秒检查一次 Agent 状态
